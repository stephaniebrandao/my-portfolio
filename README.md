# 🚀 My Personal Portfolio

This is my personal portfolio, designed to showcase my projects, skills, and contact information in a clean, modern, and professional way.

## 🛠 Tech Stack
* **React**: Library for building the user interface.
* **TypeScript**: Ensuring robust and type-safe code.
* **Vite**: Fast and efficient build tool.
* **CSS/Tailwind**: Styled for a responsive and modern experience.

## ✨ Features
* **Minimalist & Responsive Design**: Works seamlessly on all devices.
* **Quick Links**: Easy access to LinkedIn, GitHub, and email.
* **Project Showcase**: A curated list of my work with links to GitHub and live demos.
* **CV Download**: Direct access to my updated resume.

## 🚀 How to Run Locally

1. **Clone the repository:**
   git clone https://github.com/stephaniebrandao/my-portfolio.git

2. **Navigate to the folder:**
   cd my-portfolio

3. **Install dependencies:**
   npm install

4. **Start the server:**
   npm run dev
